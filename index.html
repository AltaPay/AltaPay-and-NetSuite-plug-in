Altapay and NetSuite plug-in

Receive secure payments in your NetSuite web shop

Download .zip Download .tar.gz View on GitHub

Through the partnership of AltaPay and NetSuite, we have made it much easier for you as merchant/developer to receive secure payments in your web shop. AltaPay is fully integrated with NetSuite platform via a bundle. All you have to do is to install the bundle, which will only take a few minutes to complete.

Prerequisites

We highly recommend to gather all the below information before starting the installation.

1. Prerequisites for installing the bundle

  • The bundle ‘Payment Gateways’ (ID 47196) needs to be installed on the merchant NetSuite account.
  • Be aware that due to NetSuite limitations consumers do not get informed if the transaction declines/fails. Only the merchant will be notified. This is only the case if NetSuite is used as a frontend.

2. Prerequisites for configuration of AltaPay bundle. AltaPay provides these.

AltaPay credentials

  • Username
  • Password

AltaPay gateway information

  • Terminal
  • Gateway

3. Prerequisites for fraud detection

  • Agreement with external fraud partner such as ACI ReD or MaxMind.
